Netflix Data Analysis using IBM SPSS Statistics
Overview

This project analyzes the Netflix Movies and TV Shows dataset using IBM SPSS Statistics to understand the distribution of content on the platform. The analysis focuses on identifying patterns in content type, ratings, and country-wise production trends.

The goal of this project is to demonstrate data analysis, statistical techniques, and data interpretation skills using SPSS.

Dataset

Dataset Name: Netflix Movies and TV Shows

The dataset contains information about Netflix content including:

Title

Content Type (Movie / TV Show)

Country of Production

Release Year

Rating

Duration

Date Added to Netflix

Dataset File: netflix_titles.csv

Tools & Technologies

IBM SPSS Statistics

CSV Dataset

Statistical Analysis Techniques

Data Visualization

Analysis Performed
1. Frequency Analysis

Used frequency statistics in SPSS to determine the distribution of content types and ratings.

2. Crosstab Analysis

Performed cross-tabulation to analyze relationships between:

Content type and rating

Content type and country

3. Content Type Distribution

Analyzed the proportion of Movies vs TV Shows available on Netflix.

4. Rating Distribution

Examined how Netflix content is distributed across different age ratings such as:

TV-MA

TV-14

PG

PG-13

5. Country-Wise Analysis

Identified which countries produce the most Netflix content.

Key Insights

Movies dominate Netflix content, making up approximately 69% of the catalog.

TV Shows account for about 31% of the total content.

The United States produces the largest share of Netflix content.

Netflix content production has increased significantly after 2015.

Most content falls under TV-MA and TV-14 ratings, indicating a focus on mature audiences.
