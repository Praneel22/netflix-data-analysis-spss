# Netflix Data Analysis using IBM SPSS Statistics

Dataset: Netflix Movies and TV Shows
Tool Used: IBM SPSS Statistics

## Objective

Analyze Netflix content distribution, ratings, and country trends.

---

## Overview

This project analyzes the **Netflix Movies and TV Shows dataset** using **IBM SPSS Statistics**.

The goal of this project is to explore the distribution of **content types, ratings, and country-wise production** using statistical analysis and visualization.

---

## Tools Used

* IBM SPSS Statistics
* Netflix Dataset (CSV)

---

## Analysis Performed

* Frequency Analysis
* Crosstab Analysis
* Content Type Distribution
* Rating Distribution
* Country-wise Analysis

---

## Key Insights

* Movies represent about **69%** of Netflix content.
* TV Shows represent about **31%** of Netflix content.
* The **United States** produces the highest amount of Netflix content.
* Content production **increased significantly after 2015**.
